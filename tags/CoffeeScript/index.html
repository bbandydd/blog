CoffeeScript 介紹

什麼是CoffeeScript？

  • 借用Python和Ruby的優點
  • 以較簡潔的方式編寫JavaScript
    • 以縮排取代大括號
    • List Comprehension
    • if modifier, 可以將if寫在後面，也有unless
    • 呼叫方法或邏輯判斷的時候，可以視情況省略小括號
    • 結尾不需要分號
    • 不用特別寫return
