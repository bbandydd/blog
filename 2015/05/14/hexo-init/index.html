Hexo 標記語法

什麼是Hexo?

Hexo 是一個快速、簡單且強大的網誌框架。Hexo 使用 Markdown（或其他渲染引擎）解析您的文章，並在幾秒鐘內，透過漂亮的主題產生靜態檔案。

Why Hexo ?

使用 Hexo 建立 Blog 的好處有以下：

  • 文章裡面可以放上 Coge 並加上 highlight 高亮語法顯示。
  • 使用 macdown 語言快速編輯，筆記、Blog 一次搞定。
  • 可建立於 GitHub
  • 使用 Node.js 建立
